
# Ecommerce Sales Analytics Python

## 📊 Project Overview
This project is an end-to-end **data analysis case study** on e-commerce orders using **Python, NumPy, Pandas, Matplotlib, and Seaborn**.  
It demonstrates how raw sales data can be cleaned, analyzed, and visualized to generate actionable business insights.

---

## ⚙️ Features
- **NumPy Operations**
  - Mean & median of quantities  
  - Maximum & minimum order prices  
  - Revenue calculation per order  
  - Normalization of quantity values  

- **Pandas Data Analysis**
  - Load and explore datasets  
  - Unique product identification  
  - Customer order frequency  
  - Average order value by region  
  - Date-range filtering  
  - Sorting, grouping, and aggregation  
  - Handling missing values & duplicates  
  - Removing invalid/negative entries  

- **Visualizations (Matplotlib & Seaborn)**
  - Line chart: Sales over time  
  - Bar chart: Sales by region  
  - Pie chart: Product sales distribution  
  - Line chart: Orders per day  
  - Boxplot: Total amount by region  

---

## 📂 Dataset
The dataset used is `ecommerce_orders.csv`, containing fields such as:  
- **OrderID, CustomerID, Product, Quantity, Price, OrderDate, Region, PaymentMode, TotalAmount**
